# Music Console Learning Repo

This repo is dedicated to rebuilding and understanding the ingestion and SQL modeling logic from the Music KPI Console project.

## Goal

The goal is to practice writing the project logic manually with minimal AI help.

## Learning Scope

1. Python ingestion scripts
2. Raw CSV to cleaned staging CSV
3. Data cleaning functions
4. Date parsing and validation
5. SQL staging, intermediate, and mart models
6. Snowflake upload flow later

## Rule

Full raw datasets are not committed to GitHub. They stay local inside `data/raw/`.
